```rust
// Never put on GC heap — use Box::into_raw pattern (same as SqliteConn).
struct MeshRequestData {
    method: String,
    url: String,
    headers: Vec<(String, String)>,
    body: Option<String>,
    timeout_ms: Option<u64>,
    is_json: bool,
    query_params: Vec<(String, String)>,
}

// MeshClientResponse: GC-allocated #[repr(C)] struct with 3 fields.
// Layout: { status: i64, body: *mut u8, headers: *mut u8 }
// Field order MUST match the MirStructDef registered in lower.rs.
#[repr(C)]
pub struct MeshClientResponse {
    pub status: i64,
    pub body: *mut u8,    // *mut MeshString (GC-allocated)
    pub headers: *mut u8, // *mut MeshMap (GC-allocated)
}
```

Implement these functions:
- `mesh_http_build(method: *const MeshString, url: *const MeshString) -> u64` — allocates MeshRequestData, normalizes method to lowercase, returns Box::into_raw as u64
- `mesh_http_header(handle: u64, key: *const MeshString, val: *const MeshString) -> u64` — mutates headers vec, returns same handle
- `mesh_http_body(handle: u64, body: *const MeshString) -> u64` — sets body field, returns handle
- `mesh_http_timeout(handle: u64, ms: i64) -> u64` — sets timeout_ms, returns handle
- `mesh_http_query(handle: u64, key: *const MeshString, val: *const MeshString) -> u64` — appends to query_params, returns handle
- `mesh_http_json(handle: u64, body: *const MeshString) -> u64` — sets body + is_json=true, returns handle
- `mesh_http_send(handle: u64) -> *mut u8` — executes the request, returns Result<HttpResponse, String>

For `mesh_http_send`:
1. Reconstruct `&mut MeshRequestData` from handle.
2. Build `Agent::config_builder()` with optional timeout_ms (apply per-request if set; default 30s global).
3. Start the request using `agent.method(&data.method, &url_with_query)`.
4. Chain headers: `.header(key, val)` for each.
5. Call `.http_status_as_error(false)` — CRITICAL per locked decision (Ok for all status codes).
6. Send: `.call()` for GET/HEAD, or `.send(body_bytes)` for requests with body.
7. On Ok(response):
   - Extract `status: i64 = response.status() as i64`
   - Extract body: `let body_str = response.body_mut().read_to_string().unwrap_or_default()` — NOTE: ureq 3 API, NOT into_string()
   - Extract headers: iterate `response.headers()` building a MeshMap via `mesh_map_new_typed(1)` + `mesh_map_put`
   - Allocate MeshClientResponse on GC: `mesh_gc_alloc_actor(size, align) as *mut MeshClientResponse`
   - Set fields: `(*ptr).status = status`, `(*ptr).body = body_mesh_ptr`, `(*ptr).headers = headers_map_ptr`
   - Return `alloc_result(0, ptr as *mut u8) as *mut u8`
8. On Err(e): map error to "TIMEOUT: ...", "DNS_FAILURE: ...", "TLS_ERROR: ..." format strings. Return `alloc_result(1, err_str_mesh_ptr) as *mut u8`.

Update ureq 2 compat functions (`mesh_http_get`, `mesh_http_post`) to use ureq 3 API:
- `response.body_mut().read_to_string()` instead of `response.into_string()`
- Use a default Agent (no config needed for simple requests)

**Note:** Claude should check the actual ureq 3 API for how to send a body (it may be `.send_bytes()` or `.send_form()` etc. — use `request.send(body.as_bytes())` for the generic body send).
